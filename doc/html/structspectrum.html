Detailed Description

A spectrum, consisting of a gsl_array containing the data, a start value (center wave length of lowest bin), a bin size, and warn flags, which may be

  • SPC_W_CONTAM -- Spectrum is contaminated
  • SPC_W_BORDER -- Spectrum lies partially beyond the image borders.
