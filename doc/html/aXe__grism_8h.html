ap_pixel* make_spc_table ( object *const   ob,
const int  beamorder,
int *const   flags 
)

computes a table of aperture pixels, i.e. of tuples containing the source coordinates, the distance to the spectrum trace, the path length along the trace, and the intensity. The end of the table is marked with an ap_pixel->x==-1. This routine can do subsampling, whereby each pixel is divided into n_sub*n_sub smaller pixels. If n_sub==1, a special (faster) handling is enabled.

Parameters:
ob the object struct to examine
beamorder the order of the spectrum to examine
flags warning flags like in the warning field of the spectrum structure. This has to be passed in initialized.
