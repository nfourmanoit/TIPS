00652

00902  *     this subroutine uses householder transformations with column
00903  *     pivoting (optional) to compute a qr factorization of the

00905  *     matrix q, a permutation matrix p, and an upper trapezoidal

00925  *      a is an m by n array. on input a contains the matrix for
